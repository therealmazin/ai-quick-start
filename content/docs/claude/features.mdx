# Claude Features & Tools

Claude has several powerful features that make it more than just a chatbot. Here's what you can do.

## Web Search

Claude can search the internet to find current information.

**What it does:**
- Finds up-to-date information from the web
- Provides citations and sources
- Great for fact-checking and current events

**How to use it:**
- Just ask about current events or recent information
- Claude automatically searches when needed
- You can also say "search for..." to be explicit

<Callout type="info">
Web search with citations is available to all users (currently in the US).
</Callout>

**Example prompts:**
- "What are the latest developments in AI regulation?"
- "Search for recent studies on remote work productivity"
- "Find current admission deadlines for [university]"

---

## Research Mode

Claude's Research feature does deep investigation on topics and delivers structured, cited reports.

**What it does:**
- Searches multiple sources and synthesizes information
- Creates structured reports with citations
- Can search the web AND your connected documents (Gmail, Google Docs)
- Much faster than competitors (often under 1 minute)

**Two levels:**

| Mode | Time | Best for |
|------|------|----------|
| **Research** | Under 1 minute | Quick answers with sources |
| **Advanced Research** | 5-45 minutes | Deep, complex investigations |

**How to use it:**

<Steps>
<Step>
Look for the **Research** button or mode selector
</Step>
<Step>
Type your research question
</Step>
<Step>
Claude searches, analyzes, and compiles results
</Step>
<Step>
Get a structured report with citations
</Step>
</Steps>

**Example prompts:**
- "Research best practices for faculty development programs"
- "Find and compare different project management methodologies"
- "What does research say about effective online teaching strategies?"

<Callout type="info">
Regular Research is on Pro plans. Advanced Research is on Max, Team, and Enterprise plans.
</Callout>

## Artifacts

Artifacts are Claude's standout feature. When Claude creates something substantial (documents, code, diagrams), it appears in a separate window you can edit, save, and share.

**What it does:**
- Creates standalone content in a dedicated window
- Supports documents, code, diagrams, even interactive apps
- Keeps version history (never lose your work)
- Can be published and shared with anyone

**Types of Artifacts:**

| Type | Examples |
|------|----------|
| **Documents** | Reports, articles, guides, formatted text |
| **Code** | Working code snippets, interactive components |
| **Diagrams** | Flowcharts, mind maps, org charts, timelines |
| **Web pages** | Complete HTML/CSS/JavaScript pages |
| **Interactive apps** | Calculators, quizzes, simple tools |

**How to use it:**
- Ask Claude to create something substantial
- It automatically opens in the Artifact panel (right side)
- Edit, iterate, and refine with follow-up prompts
- Click "Publish" to share with others

<Callout type="info">
Artifacts are created automatically when content is over ~15 lines and can stand alone.
</Callout>

**Example prompts:**
- "Create a flowchart showing our approval process"
- "Build a simple quiz about workplace safety"
- "Write a project proposal template I can reuse"
- "Make an organizational chart for a small team"

## Mermaid Diagrams

Claude can create professional diagrams using Mermaid (a simple diagram language).

**Types of diagrams:**
- Flowcharts
- Sequence diagrams
- Gantt charts (timelines)
- Mind maps
- Org charts
- Entity relationship diagrams

**How to use it:**
- Ask Claude to "create a diagram" or "make a flowchart"
- It generates a Mermaid diagram in an Artifact
- The diagram renders visually

**Example prompts:**
- "Create a flowchart for the student enrollment process"
- "Make a Gantt chart for a 3-month project"
- "Draw a mind map of topics for our training program"

---

## Analysis Tool

Claude can analyze data and create interactive visualizations.

**What it does:**
- Analyzes datasets you upload
- Identifies patterns and trends
- Creates charts and dashboards
- Runs JavaScript to process data

**How to use it:**
- Upload a data file (CSV, Excel)
- Ask Claude to analyze it
- Get insights and visualizations in an Artifact

**Example prompts:**
- "Analyze this survey data and show me the key findings"
- "Create a chart showing attendance trends"
- "What patterns do you see in this spreadsheet?"

## Projects

Projects let you organize your work and give Claude persistent context. (Covered in detail in the [Projects guide](/docs/claude/projects))

**Quick overview:**
- Create workspaces for different tasks
- Upload documents Claude can always reference
- Set custom instructions for each project
- Keep related conversations together

<Callout type="info">
Projects are available on Pro and Team plans.
</Callout>

---

## File Uploads

Upload documents for Claude to read, analyze, and discuss.

**Supported files:**
- PDF documents
- Word documents (.docx)
- Text files (.txt)
- HTML files
- Spreadsheets (.csv)
- Code files
- EPUB books

**Limits:**
- Single file: up to 30MB
- Total content must fit in context (~500 pages worth)

**How to use it:**
- Click the attachment icon
- Upload your file
- Ask questions about it

**Example prompts:**
- "Summarize the key points in this document"
- "What are the main recommendations in this report?"
- "Compare these two documents and highlight the differences"

---

## Integrations

Claude can connect to your work tools for deeper context.

**Available integrations:**
- **Google Docs** - Search and reference your documents
- **Gmail** - Search your emails for context
- **Google Calendar** - See your schedule
- More coming soon

**How it helps:**
- Research mode can search across your connected apps
- Claude understands your work context better
- Find information without leaving Claude

<Callout type="info">
Integrations are available on Pro, Max, Team, and Enterprise plans.
</Callout>
