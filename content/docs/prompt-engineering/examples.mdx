# Ready-to-Use Prompt Templates

Here are prompts you can copy and customize for common tasks. Just replace the parts in `[brackets]` with your own information.

## Email Writing

### Professional Email

```
Write a professional email to [recipient] about [topic].

Context: [brief background]

Tone: Professional but friendly
Length: Keep it concise (3-4 paragraphs max)

Key points to include:
- [Point 1]
- [Point 2]
- [Point 3]
```

### Follow-Up Email

```
Write a polite follow-up email.

Original context: I sent an email about [topic] on [date] and haven't received a response.

Goal: Gently remind them and ask for an update
Tone: Friendly, not pushy
```

### Thank You Email

```
Write a thank you email to [person] for [what they did].

I want to express genuine appreciation while keeping it professional.
Keep it brief but warm.
```

## Document Summarization

### Summarize a Document

```
Please summarize this document.

Focus on:
- Main points
- Key decisions or recommendations
- Action items (if any)

Format: Bullet points
Length: No more than [X] bullet points

[Paste your document or upload the file]
```

### Meeting Notes Summary

```
Here are notes from a meeting. Please:

1. Summarize the key discussion points
2. List any decisions that were made
3. Extract action items with who is responsible

Format as clear sections with headers.

[Paste meeting notes]
```

## Writing Assistance

### Improve My Writing

```
Please improve this text:

[Paste your text]

Goals:
- Make it clearer and more concise
- Fix any grammar issues
- Keep the same general meaning
- Maintain a [formal/casual/professional] tone
```

### Change the Tone

```
Rewrite this text to be more [formal/casual/friendly/professional]:

[Paste your text]

Keep the same information, just change the tone.
```

### Proofread

```
Please proofread this text for:
- Grammar and spelling errors
- Awkward phrasing
- Consistency issues

Highlight what you changed and why.

[Paste your text]
```

## Research & Learning

### Explain a Concept

```
Explain [concept/topic] to me.

My background: [beginner/intermediate/expert in this field]

Please:
- Use simple language
- Give a real-world example
- Keep it to about [X] paragraphs
```

### Compare Options

```
Help me compare [Option A] vs [Option B] for [purpose/use case].

Please create a comparison table covering:
- Key features
- Pros and cons
- Best use cases
- [Any specific criteria you care about]
```

### Research Summary

```
Give me an overview of [topic].

Include:
- What it is (brief definition)
- Why it matters
- Key things to know
- Common misconceptions (if any)

Keep it accessible for someone new to this topic.
```

## Planning & Brainstorming

### Generate Ideas

```
I need ideas for [what you're planning].

Context: [brief background]
Constraints: [any limitations - budget, time, audience, etc.]

Please give me [X] different ideas, ranging from simple to creative.
For each idea, briefly explain why it could work.
```

### Create an Outline

```
Help me create an outline for [document/presentation/project].

Purpose: [what you're trying to achieve]
Audience: [who will read/see this]
Key points I want to cover:
- [Point 1]
- [Point 2]
- [Point 3]

Format: Hierarchical outline with main sections and sub-points
```

### Action Plan

```
Help me create an action plan for [goal/project].

Current situation: [where you are now]
Goal: [where you want to be]
Timeline: [when you need to finish]

Please break this into clear, actionable steps.
```

## Data & Analysis

### Analyze This Data

```
Please analyze this data:

[Paste data or describe what you're sharing]

I'm looking for:
- Key patterns or trends
- Notable outliers
- Summary statistics
- Recommendations based on what you see
```

### Create a Table

```
Create a table organizing [information about what].

Columns should include:
- [Column 1]
- [Column 2]
- [Column 3]

[Provide the information or context]
```

## Tips for Using These Templates

<Callout type="info">
These templates are starting points. Customize them based on your needs!
</Callout>

### How to Customize

1. **Replace bracketed text** with your specific information
2. **Add or remove sections** based on what you need
3. **Adjust the tone** to match your context
4. **Be specific** about what you want

### When Responses Aren't Right

If the AI's response isn't quite what you wanted:

- **Too long?** Say: "Make this shorter" or "Summarize in 3 sentences"
- **Too formal?** Say: "Make this more casual and friendly"
- **Missing something?** Say: "Add information about [topic]"
- **Wrong focus?** Say: "Focus more on [aspect] and less on [aspect]"

### Build Your Own Templates

Once you find prompts that work well for you, save them! Create your own collection of templates for tasks you do regularly.
