## Quick Privacy Tip

Before you start using any AI tool, we recommend turning off data training. This keeps your conversations private.

- **ChatGPT**: [Turn off "Improve the model for everyone"](/docs/chatgpt/privacy)
- **Claude**: [Turn off "Help improve Claude"](/docs/claude/privacy)
