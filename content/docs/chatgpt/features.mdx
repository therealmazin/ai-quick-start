# ChatGPT Features & Tools

ChatGPT isn't just a chatbot - it has powerful tools built in. Here's what you can do with them.

## Web Search

ChatGPT can search the internet in real-time to find current information.

**What it does:**
- Finds up-to-date information (news, prices, events)
- Provides source links so you can verify
- Great for questions about recent events

**How to use it:**
- Just ask a question about current events or real-time data
- ChatGPT automatically decides when to search
- You can also say "search the web for..."

<Callout type="info">
Web search is available to all users, including free accounts.
</Callout>

**Example prompts:**
- "What's the weather in Bahrain today?"
- "Search for the latest news about AI regulations"
- "What are the current admission requirements for [university]?"

## Deep Research

Deep Research is like having a research assistant. It spends 5-30 minutes browsing hundreds of sources to create a comprehensive report.

**What it does:**
- Researches complex topics thoroughly
- Reads and analyzes hundreds of web pages
- Creates detailed reports with citations
- Can export to PDF or Word

**How to use it:**

<Steps>
<Step>
Click the **+** button in the chat
</Step>
<Step>
Select **Deep Research** from the menu
</Step>
<Step>
Describe what you want researched (be specific!)
</Step>
<Step>
Wait 5-30 minutes for the report
</Step>
<Step>
Download as PDF or Word when done
</Step>
</Steps>

**Best for:**
- Competitive analysis
- Literature reviews
- Market research
- Comparing products or services
- Policy research

<Callout type="warn">
Deep Research has monthly limits:
- **Free**: 5 queries (lightweight version)
- **Plus**: 10 full + 15 lightweight per month
- **Pro**: 125 + 125 per month
</Callout>

**Example prompts:**
- "Research the pros and cons of different learning management systems for universities"
- "Create a report on best practices for employee onboarding programs"
- "Analyze trends in online education over the past 5 years"

## Canvas

Canvas is a collaborative workspace where you and ChatGPT can write and edit together, side by side.

**What it does:**
- Opens a separate editing window next to your chat
- Lets you highlight specific sections for feedback
- Suggests edits, adjusts length, changes reading level
- Keeps version history (you can undo changes)
- Exports to PDF, Word, or Markdown

**How to use it:**

<Steps>
<Step>
Type **/canvas** in chat, or say "use canvas" or "open canvas"
</Step>
<Step>
ChatGPT opens a split-screen editor
</Step>
<Step>
Write or paste your content
</Step>
<Step>
Highlight text and ask for specific changes
</Step>
<Step>
Use the toolbar for quick actions (edit, shorten, polish)
</Step>
</Steps>

**Quick actions in Canvas:**
| Action | What it does |
|--------|--------------|
| Suggest edits | Get inline editing suggestions |
| Adjust length | Make content longer or shorter |
| Reading level | Simplify or make more advanced |
| Add polish | Final cleanup and refinement |
| Add emojis | Make it more engaging |

<Callout type="info">
Canvas opens automatically when ChatGPT generates content longer than 10 lines.
</Callout>

**Best for:**
- Drafting and editing documents
- Writing emails, reports, articles
- Getting line-by-line feedback
- Collaborative writing projects

## Code Interpreter (Advanced Data Analysis)

ChatGPT can write and run Python code to analyze your data.

**What it does:**
- Runs calculations and analysis
- Creates charts and visualizations
- Processes spreadsheets and datasets
- Generates downloadable files

**How to use it:**
- Upload a file (CSV, Excel, etc.)
- Ask ChatGPT to analyze it
- It writes and runs code automatically
- You see the results and can download outputs

**Example prompts:**
- "Analyze this spreadsheet and show me the trends"
- "Create a chart showing sales by month"
- "Calculate the average and standard deviation"

<Callout type="info">
Available on Plus and Pro plans.
</Callout>

---

## Image Generation (DALL-E)

ChatGPT can create images from your descriptions using DALL-E 3.

**What it does:**
- Creates custom images from text descriptions
- Good for illustrations, diagrams, creative images
- Generates multiple variations

**How to use it:**
- Just describe the image you want
- Say "create an image of..." or "generate a picture of..."
- ChatGPT creates it right in the chat

**Example prompts:**
- "Create an illustration of a diverse team collaborating in a modern office"
- "Generate a simple diagram showing the research process"
- "Make a professional banner image for a workshop about AI"

<Callout type="info">
Available on Plus and Pro plans. Free users have limited access.
</Callout>

---

## Voice Mode

Talk to ChatGPT like you're having a real conversation.

**What it does:**
- Speak naturally instead of typing
- Hear ChatGPT respond with voice
- Use your camera to show things and discuss them (mobile)

**How to use it:**
- Click the microphone/headphone icon
- Start talking
- ChatGPT responds with voice

**Best for:**
- Hands-free use
- Practicing conversations or presentations
- Quick questions when typing is inconvenient
- Language practice

<Callout type="info">
Basic voice is available to all users. Advanced Voice Mode (more natural conversations) is on paid plans.
</Callout>

---

## File Uploads

Upload documents for ChatGPT to read and analyze.

**Supported files:**
- PDF documents
- Word documents (.docx)
- Excel spreadsheets (.xlsx)
- PowerPoint presentations (.pptx)
- Images (for analysis)
- Text files
- Code files

**How to use it:**
- Click the paperclip/attachment icon
- Select your file
- Ask questions about it

**Example prompts:**
- "Summarize this document"
- "What are the key points in this PDF?"
- "Extract the action items from these meeting notes"
